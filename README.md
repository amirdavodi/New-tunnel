# 🚀 Omni-Path Tunnel (Gost V3) - 2026 Edition

یک اسکریپت تعاملی و هوشمند برای برقراری تانل امن و پایدار بین دو سرور (ایران و خارج) با استفاده از Gost V3. این پروژه برای دور زدن محدودیت‌های اینترنت و بهینه‌سازی ترافیک طراحی شده است.

## ✨ ویژگی‌های برجسته
* **نصب کاملاً تعاملی:** بدون نیاز به دانش فنی، اسکریپت سوالات را می‌پرسد و تنظیمات را انجام می‌دهد.
* **بهینه‌سازی BBR:** فعال‌سازی خودکار پروتکل BBR برای افزایش سرعت و کاهش پینگ.
* **امنیت بالا:** استفاده از مسیر اختصاصی (`path`) و نام کاربری/رمز عبور برای تانل.
* **Reverse Tunnel (rtcp):** انتقال ترافیک به صورت معکوس برای پایداری بیشتر در شبکه‌های محدود شده.
* **مدیریت با Systemd:** اجرای تانل به عنوان یک سرویس سیستمی که با ریبوت شدن سرور، خودکار اجرا می‌شود.
